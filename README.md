# Premium React Portfolio
https://sanketnanera28.github.io/Portfolio-/#/

A modern, responsive portfolio website built with React 19 and Vite. This project features a full administrative dashboard for managing content dynamically.

## Features
- **Project Showcase**: Display your work with tech tags and live links.
- **Certificates & Qualifications**: Showcase your professional achievements and education.
- **Admin Dashboard**: Manage all your data (Projects, Certificates, Qualifications, Messages) through a secure interface.
- **Contact Form**: Integrated messaging system with local persistence.
- **Modern UI/UX**: Premium aesthetic with glassmorphism, gradients, and smooth animations.

## How to Run

### 1. Prerequisite
Ensure you have [Node.js](https://nodejs.org/) installed on your system.

### 2. Setup
Open your terminal and navigate to the project directory:
```bash
cd portfolio
```

### 3. Install Dependencies
```bash
npm install
```

### 4. Run Development Server
```bash
npm run dev
```
Once started, open the URL provided in the terminal (e.g., `http://localhost:5173`).
